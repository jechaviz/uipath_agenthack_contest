# UiPath Workflow Assets

This folder contains the proposed UiPath implementation blueprint for Agentic Incident Ops.

- `agentic_incident_ops.case_blueprint.json` is a machine-readable case design.
- The narrative workflow is documented in `../docs/UIPATH_WORKFLOW_PROPOSAL.md`.

The final production build should recreate these stages in UiPath Maestro Case and connect each integration through UiPath API Workflows, robots, Agent Builder agents, and a coded agent where needed.
